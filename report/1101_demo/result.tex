		\item 感測器與機板的連接問題
		\item 供電問題
		\item 讀取資料的程式
		\item 絕緣密閉

		\item 詳細定義食物的狀態（ ex. 如何為腐敗？），並使之量化，能放入實驗設計
	\end{itemize}

		\item 欲先解決感測器連接問題（目前考慮重買器材？）

		\item 設計肉類判別流程，該流程需可以明確判定肉類是否已腐敗
	\end{itemize}
